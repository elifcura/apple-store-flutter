# Apple Store Katalog Uygulaması

Bu proje, Flutter ile geliştirilmiş, kullanıcıların ürünleri inceleyebileceği, detaylarını görebileceği ve sepete ekleyebileceği modern bir e-ticaret katalog arayüzüdür. 


##  Proje Özellikleri
* **Ürün Listeleme:** Kategorize edilmiş ürünlerin ana sayfada sergilenmesi.
* **Detay Sayfası:** Ürünlere ait geniş açıklamalar ve büyük görseller.
* **Sepet Sistemi:** Ürünlerin sepete eklenmesi, sepet sayısının anlık güncellenmesi ve sepetten ürün silme fonksiyonu.
* **Yerel Varlıklar (Assets):** Uygulama görselleri proje içerisinde yerel olarak barındırılmaktadır.

##  Kullanılan Teknolojiler & Sürüm
* **Flutter:** 3.x.x (En güncel kararlı sürüm)
* **Dart:** 3.x.x
* **IDE:** Visual Studio Code

## Çalıştırma Adımları
Projeyi yerel makinenizde çalıştırmak için şu adımları izleyin:

1. Bu depoyu klonlayın: 
   `git clone https://github.com/elifcura/apple-store-flutter.git`
2. Proje dizinine gidin: 
   `cd mini_katalog`
3. Gerekli paketleri yükleyin: 
   `flutter pub get`
4. Uygulamayı çalıştırın: 
   `flutter run -d chrome` (veya bağlı cihazınızda)
